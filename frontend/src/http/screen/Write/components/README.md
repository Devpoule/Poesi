# Write Components

UI components used by the Write screen.

- MoodBadge: Color dot for mood selection.
- SaveIndicator: Draft status toggle with tooltip.
- WriteAccessGate: Login/register prompt for guests.
- WriteActions: Save/publish action row.
- WriteBackdrop: Decorative mood background shapes.
- WriteEditor: Full write experience for authenticated users.
- WriteEditorCard: Card that wraps fields and actions.
- WriteFields: Title/body inputs with mood-aware styling.
- WriteHeader: Page header with title and draft indicator.
- WriteMoodPanel: Mood selector and description panel.
