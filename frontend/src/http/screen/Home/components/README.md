# Home Components

UI components that render the home experience.

- HomeHero: Top hero card with primary actions.
- HomeHighlights: Value cards under the hero.
- HomeRitual: Daily prompt card.
- HomeMoodSection: Mood categories grid.
- HomeSymbolSection: Symbol categories grid.
- HomeRecentPoems: Recent poems list with filters.
- HomePoemCard: Card for a single poem preview.
- HomeSectionHeader: Title + hint + optional action link.
- HomeFilterChip: Mood filter chip for the recent poems list.
- HomeSymbolMark: Glyph renderer for symbol cards.
- HomePortal: Closing call-to-action panel.
