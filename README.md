# POESI

POESI is a minimalist poetic platform focused on short texts, moods, colors
and symbolic totems.

## Purpose
POESI allows authors to write short poetic texts and associate them with a mood
(color, totem, relic), creating a curated and immersive reading experience.

## Core concepts
- Authors write short poems
- Each poem has a mood (color)
- Totems and relics symbolize progression and recognition
- Minimalist API-first backend

## Tech stack
- PHP 8.x
- Symfony
- Doctrine ORM
- JSON API

## Architecture overview
This project follows a clean, domain-oriented architecture with clear
separation between:
- Domain logic
- Application services
- Infrastructure
